**Ответ:**

Выберите действие в меню ниже или используйте команды.
```

**Callback:** `tg:health`

**Формат ответа:**
```

Статус: 🟢 ACTIVE | Uptime: 2d 5h 30m
Версия: v2.1.0

💰 Баланс (Unified)
├ Total: $10,250.00
└ Available: $8,500.00 (USDT)

├ USDT: 8500.0000 (free: 8500.0000)
├ BTC: 0.0500 (free: 0.0300)
└ ETH: 0.5000 (free: 0.0000)

� Активные позиции (2)
🟩 ETH: +1.56% | $3,250.00
🟥 SOL: -0.80% | $178.50

💹 Арбитраж (1)
• BTC(Arb): +$8.50 funding

Последний трейд: 25 мин. назад
```

**Что показывает:**
- **Статус** — режим (🟢ACTIVE / 🟡RISK_ONLY / 🔴PAUSED) + uptime + версия

- **Позиции** — компактно с 🟩/🟥 PnL индикаторами
- **Арбитраж** — только если есть открытые (до 3 позиций)
- **Последний трейд** — время с момента последнего закрытия

**Команда:** `/regime`
**Callback:** `tg:regime`

**Формат ответа (ЗЕЛЕНЫЙ):**
```
� Рыночный режим: BTC/USDT (4H)
Статус: 🟢 ЗЕЛЕНЫЙ (Торговля разрешена)

Показатели:
├ Цена: 97,500.00 (Target > 85,200.00) ✅

└ ATR Spike: 1.15 (Limit < 2.0) ✅

Обновлено: 2026-02-09 12:00:00
```

**Формат ответа (КРАСНЫЙ):**
```
� Рыночный режим: BTC/USDT (4H)
Статус: 🔴 КРАСНЫЙ (Торговля заблокирована)

Показатели:
├ Цена: 82,000.00 (Target > 85,200.00) ❌
├ RSI14: 55.00 (Target > 50) ✅
└ ATR Spike: 2.45 (Limit < 2.0) ❌

⛔️ Режим КРАСНЫЙ: запрет новых сделок

Обновлено: 2026-02-09 12:00:00
```

**Что показывает:**
- Статус светофора с эмодзи и описанием
- Каждое условие с `<code>` значениями и ✅/❌
- **ATR Spike ratio** — отношение текущего ATR к среднему (блокирует при > 2.0)
- Визуальные индикаторы `├ └` для дерева условий
- Причина запрета (если КРАСНЫЙ)
- Время последней проверки

## ⚙️ Настройки

**Команда:** `/settings`
**Callback:** `tg:settings`

**Формат ответа:**
```
⚙️ Настройки

**Кнопки (1 в ряд):**

| Кнопка | Callback | Описание |
|--------|----------|----------|
| 🤖 Авто-торговля: ON | `tg:toggle_auto_trading` | Вкл/выкл авто-торговли |

## 🔒 Безопасность

- **Whitelist пользователей:** только `TELEGRAM_ALLOWED_USER_IDS` из конфига
- **Проверка на каждый запрос:** и команды, и callback-кнопки
- **Forbidden:** неавторизованные пользователи получают `"Forbidden"` alert
- **Rate limiter:** макс. 20 исходящих сообщений в минуту (защита от бана Telegram)
- **PANIC SELL:** двухшаговое подтверждение

## 🔄 Автоматические уведомления бота

Бот отправляет сообщения автоматически (если `notifications_enabled = True`):

| Событие | Пример сообщения |
|---------|------------------|
| Открытие позиции | `✅ Открыта позиция ETHUSDT: 0.5 @ $3200 | TP=$3360 SL=$3136` |
| Закрытие позиции | `📊 Закрыта ETHUSDT: PnL=$25.00 (+1.56%) | Причина: trailing_stop` |
| Trailing Stop сдвинут | `🔄 SL по ETHUSDT подтянут до 3280.0000 (Прибыль: 3.50%)` |
| Breakeven активирован | `🔒 Breakeven ETHUSDT: SL → $3203.20` |
| DCA докупка | `📥 DCA ETHUSDT: +0.2 @ $3150 | Средняя: $3180` |
| Арбитраж открыт | `💹 Арбитраж открыт: BTCUSDT | Spot LONG + Futures SHORT` |
| Ошибка критическая | `🚨 Ошибка: Bybit API timeout` |
| Сканер нашёл пару | `🔍 Сканер: новая пара XRPUSDT (gain +5.2%)` |
| Режим изменился | `🚦 Режим: КРАСНЫЙ → торговля заблокирована` |
